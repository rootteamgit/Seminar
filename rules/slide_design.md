# スライド設計・デザインルール

PowerPoint（.pptx）でセミナースライドを制作する際のすべてのルール。pptxgenjsで生成する場合も同じシステムを適用。

---

## 🎨 デザインシステム（pptxgenjs標準）

**基準デザイン: 第12回スライド（完全踏襲）**

### スライドサイズ
```
layout: LAYOUT_WIDE
横: 13.33 inch
縦: 7.5 inch
アスペクト比: 16:9
```

### カラーパレット
| 用途 | カラー | 16進法 |
|------|--------|--------|
| 背景 | ネイビー | #0D1147 |
| アクセント1 | シアン | #00B4D8 |
| アクセント2 | ピンク | #D9369E |
| アクセント3 | ブルー | #3A5BD9 |
| カード背景 | ダークネイビー | #1A2272 |
| フッター背景 | ディープネイビー | #151A6A |
| フッターテキスト | マゼンタ | #F040B0 |
| テキスト（基本） | ホワイト | #FFFFFF |
| テキスト（サブ） | ライトブルー | #A0B4CC |

### フォント
- **メイン**: Calibri（QA環境でも正確に描画される安全フォント）

### マージン（定数）
```javascript
const ML = 0.5;   // 左マージン
const CW = 12.33; // コンテンツ幅
```

## 🏗️ 第12回デザインの構造（必ず踏襲）

```
スライド全体に以下の要素を配置:

1. 左ピンク縦バー
   x=0, y=0, w=0.18*SX, h=全高, fill=#D9369E

2. 上部ブルーライン
   x=0.18*SX, y=0, w=9.82*SX, h=0.12*SY, fill=#3A5BD9

3. 下部ブルーライン
   x=0, y=5.45*SY, w=10*SX, h=0.17*SY, fill=#3A5BD9

4. セクション番号バッジ（左上）
   x=0.45*SX, y=0.25*SY, w=0.55*SX, h=0.55*SY, fill=#D9369E

5. セクションタイトル（バッジ右）
   x=1.15*SX, y=0.27*SY, fontSize=22, bold, color=#FFFFFF

6. カード（コンテンツ用）
   fill=#1A2272 + 左ボーダー8px（シアン/ピンク/ブルーで使い分け）

7. 番号バッジ（各カード左上）
   01/02/03 形式、カード左ボーダーと同色

8. カード本文背景
   fill=#E8EEFF（薄い水色）でテキストエリアを囲む

9. フッター強調テキスト（下部）
   背景=#151A6A, テキスト=#F040B0, fontSize=12, bold
```

## 📐 スライドテンプレート

### パターン1: タイトルスライド
- 背景: #0D1147
- セクション番号バッジ（第N回）
- タイトル: 32〜42pt, bold, white
- サブタイトル: 18pt, cyan
- 区切りライン: fill=#3A5BD9
- 日時・登壇者: 12pt, #A0B4CC

### パターン2: コンテンツスライド（カード3枚）
- フレーム（左縦バー・上下ライン）
- セクションバッジ + タイトル
- カード3枚（各1.18*SY高さ）
- y位置: 1.1*SY / 2.48*SY / 3.86*SY
- フッターテキスト

### パターン3: 3列カード
- フレーム + セクションバッジ
- 横3列カード（各colW = 2.85*SX）
- フッターテキスト

### パターン4: 2列カード（比較・対比）
- フレーム + セクションバッジ
- 左右2枚のカード（各4.4*SX幅）

### パターン5: ステップカード（縦3枚）
- フレーム + セクションバッジ + サブ見出し
- STEP 1/2/3 形式のカード（各0.95*SY高さ）
- フッターテキスト

## 🔑 スライド設計の基本原則

### 1スライドの情報密度
- 1スライド最大3カードまたは5行の箇条書きまたは1メインビジュアル
- フッターには章全体のキーメッセージを1行で入れる

### テキストサイズの階層
| 要素 | サイズ |
|------|--------|
| スライドタイトル | 32〜42pt |
| セクション見出し | 22pt |
| カードヘッダー | 17〜18pt |
| 本文 | 13〜15pt |
| フッター | 12pt |

### スライドと台本の照合（必須）
- 台本の📊スライド切替マークと枚数が完全一致していること
- 作成後に必ず枚数をダブルチェックする

## 🚨 NGパターン

| NG | 理由 | 修正案 |
|----|------|--------|
| カード左ボーダーなし | 第12回テイストが崩れる | 必ず8pxボーダーを入れる |
| フッターなし | 章の締めメッセージが消える | 必ず下部にフォロー強調テキスト |
| フォントをYu GothicやAptos指定 | QA環境で崩れる | Calibri固定 |
| 背景色を#0D1B2Eにする | 旧仕様。基準は#0D1147 | 必ず#0D1147を使う |
| 1スライドにテキスト10行以上 | 視認性低下 | 2スライドに分割 |

## 🔄 pptxgenjsでの自動生成時の設定

```javascript
const prs = new pptx();
prs.layout = 'LAYOUT_WIDE'; // 13.33×7.5inch

const ML = 0.5;   // 左マージン（inch）
const CW = 12.33; // コンテンツ幅（inch）
const FF = 'Calibri'; // フォント

// スケール係数（第12回EMU座標からの変換用）
const SX = 13.33 / 10.0; // 1.333
const SY = 7.5 / 5.62;   // 1.334

const C = {
  navy:     '0D1147',
  navyCard: '1A2272',
  blue:     '3A5BD9',
  pink:     'D9369E',
  cyan:     '00B4D8',
  white:    'FFFFFF',
  textSub:  'A0B4CC',
  footerBg: '151A6A',
  footerTx: 'F040B0',
};
```
